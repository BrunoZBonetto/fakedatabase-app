## Submission Templates

### Generic Description (Short)
> Free online fake data generator with 120+ fields, 15 templates, and 7 export formats (CSV, Excel, JSON, SQL, XML, YAML). Runs in the browser, no signup required.

### Generic Description (Long)
> Fake Database is a free, browser-based tool for generating realistic fake data. It supports 120+ data fields across 19 categories (names, addresses, financial, health, vehicles, entertainment, etc.), 15 pre-built templates, and exports to CSV, Excel, JSON, SQL, XML, and YAML. Features include cross-field correlations for realistic data, custom columns with SQL types, null/error injection, and support for both US and Brazilian data. No signup, no limits, no data sent to servers.

### Tags/Keywords
fake data, mock data, test data, generator, CSV, Excel, JSON, SQL, testing, QA, development, prototype, demo, free, browser, React, TypeScript
