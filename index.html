What is Fake Database?

Fake Database is a free, browser-based tool that generates realistic fake data for developers, QA engineers, and data analysts. Unlike paid alternatives like Mockaroo ($49/month) or backend-dependent tools like Faker.js, Fake Database runs entirely in your browser — no server, no signup, no data sent anywhere.

With 120+ data fields across 19 categories, 15 quick templates, and 7 export formats, it's the most complete free fake data generator available online.

Key Features

120+ Data Fields

Names, addresses, emails, phones, financial data, health records, vehicles, entertainment, education, and more. All fields generate locale-aware data for US and Brazil.

15 Quick Templates

Fake Database supports both American English (EN-US) and Brazilian Portuguese (PT-BR) data with locale-aware formatting:

  • Phone numbers: +1 (XXX) XXX-XXXX (US) and (DDD) 9XXXX-XXXX (Brazil)

  • Dates: MM/DD/YYYY (US) vs DD/MM/YYYY (Brazil)

  • Shipping: UPS/FedEx/DHL (US) vs Correios/Jadlog/Loggi (Brazil)

Frequently Asked Questions

Is Fake Database really free?

Yes. Fake Database is 100% free with no limits. Generate up to 5,000 records per request. No signup, no credit card, no ads.

Is my data safe?

Absolutely. Fake Database runs entirely in your browser. No data is sent to any server. All generation happens locally on your device.

How is it different from Mockaroo?

Mockaroo charges $49/month for SQL export, Excel export, and custom columns. Fake Database offers all of that for free. Mockaroo also requires signup; Fake Database doesn't.

How is it different from Faker.js?

Faker.js requires a backend server (Node.js) and generates each field independently. Fake Database runs in the browser with cross-field correlations that make data internally consistent.

Can I generate Brazilian data?

Yes! Fake Database supports both US and Brazilian data, including CPF/CNPJ, Brazilian addresses, phone numbers, license plates, and culturally localized names, foods, and more.

What export formats are supported?

CSV, Excel (XLSX), JSON, SQL (CREATE TABLE + INSERT), XML, YAML, and an interactive table view.

Technology

Built with React 18, TypeScript, and Vite. Uses @dnd-kit for drag-and-drop field reordering, SheetJS for Excel export, and Web Workers for generating large datasets without blocking the UI. Deployed on Netlify. Open source on GitHub.
